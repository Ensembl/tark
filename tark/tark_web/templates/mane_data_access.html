Accessing MANE data

Note: We will have several data releases as we make progress on completing the MANE projects.

 Browse list

Browse or export a list of all MANE transcripts in Tark.

Browse matches from GRCh37 showing equivalence in 5' UTR, 3' UTR and CDS in relation to MANE Select and MANE Plus Clinical transcripts on GRCh38.

 Bulk data download

The full MANE transcript datasets can be downloaded via:

  • EBI's FTP (data in GFF and GTF formats).
  • NCBI's FTP. See the README file (data in FASTA, GFF3 and GTF formats for both Ensembl (ENST) and RefSeq (NM) identifiers).
  • GenBank files are available for RefSeq transcripts and proteins. (e.g. NM_014491.4)
  • For the GFF and GTF RefSeq annotation files:
    • Column 9 contains a “MANE Select” tag attribute in the rows associated with the mRNA, CDS and exon features.
    • Column 9 contains the matching Ensembl transcript identifier as an external database reference (Dbxref).
    • Rows in the annotation files associated with the CDS feature contain the MANE Select tag, along with the matching Ensembl protein identifier.

 Web services

The full list of MANE Select transcripts can be accessed programmatically using Tark’s REST API to GET the results in JSON format. For more information, see here.

Ensembl provides a RESTful web service for access to Ensembl data including the MANE transcripts. For example this works for Ensembl's Variant Effect Predictor (VEP) calls, enabling MANE transcript annotation to be included in VEP results, e.g.

  • Fetch variant consequences for multiple ids, including MANE Select annotations
  • Fetch variant consequences, including MANE Select annotations

 Track hubs

We have released this data as a trackhub to visualise the most recent release of MANE data as a track in genome browsers, such as hosted by Ensembl, NCBI and UCSC.

  • Via the TrackHub Registry
  • From NCBI here

 Browsers and other resources

MANE transcripts are already displayed in the following resources:

  • Ensembl, VEP and BioMart, (see the Ensembl transcript table for example)
  • UCSC on the GRCh38 (hg38) assembly, (see configuration page)
  • NCBI’s Genome Data Viewer (GDV) or Variation Viewer browsers
  • The Genome Aggregation Database (gnomAD v3 - see example)
  • ClinVar (see example)
  • DECIPHER (see example)
  • ClinGen Allele Registry (see example)
  • NCBI Entrez search
  • HGVS
