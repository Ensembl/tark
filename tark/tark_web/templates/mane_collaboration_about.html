About the MANE Collaboration

The MANE project - The Matched Annotation from the National Center for Biotechnology Information (NCBI) and the European Molecular Biology Laboratories - European Bioinformatics Institute (EMBL-EBI) (MANE) is a collaborative project that aims to converge on human gene annotation to produce a genome wide transcript set that:

  • matches GRCh38
  • includes pairs of RefSeq (NM) and Ensembl/GENCODE (ENST) transcripts that are 100% identical (5’ UTR, coding region and 3’ UTR)
  • is high-confidence based on conservation, expression and overall LRG creation process – Locus Reference Genomic – LRG sequences provide a stable genomic DNA framework for reporting variants with a permanent ID and core content that never changes.

Each transcript will retain its original identifier e.g. "ENST00000350908.9", or "NM_000088.3".

Background

Clinical data are more often reported using RefSeq (NM) transcripts, whereas large-scale genomics projects more often default to Ensembl’s ENSTs. This project will eliminate differences between RefSeq and Ensembl annotation for the MANE subset of transcripts to encourage and facilitate bi-directional exchange of data across RefSeq and Ensembl transcripts. Finally, the perfect alignment of all MANE transcripts to GRCh38 makes the set compatible with NGS-based sequencing technologies and other resources which default to GRCh38.

The MANE Select set will serve as a default set across genomics resources. Thus our effort to select one high-quality transcript at all protein-coding loci, and to have this be consistent across all genomics resources, will give a consistent starting view of biology for researchers, whether the intent is to use it for reporting variants, comparative genomics or any other endeavour. For the MANE Plus Clinical, we have identified additional transcripts as a standard critical for variant reporting.

Current and future projects

MANE Select Current project

  • This set consists of one high-quality transcript at each protein-coding locus across the genome that is representative of the biology at that gene locus and is well-supported by experimental evidence.
  • MANE Select transcripts are identified using computational methods complemented by manual review and discussion. The methods consider evidence of functional potential such as expression levels, evolutionary conservation and clinical significance.
  • This set is useful as:
    • a default for display on browsers,
    • a standard for clinical reporting,
    • a starting point for comparative or evolutionary genomics.
Note: All transcripts for a given locus should always be considered and we are certainly NOT saying that biology can be simplified to a single transcript.

MANE Plus Clinical Current Project

  • Additional transcripts per locus necessary to support clinical variant reporting, for example transcripts containing known “Pathogenic” or “Likely Pathogenic” clinical variants not reportable using the MANE Select set.
  • Note there may be additional clinically relevant transcripts in the wider RefSeq and Ensembl/GENCODE sets but not yet in MANE.
  • Collaboration with, and feedback from, the clinical community is an important aspect of the MANE project. Transcripts of clinical interest not included in our MANE sets will be considered upon request and in consultation with expert clinical scientists. Contact email mane-help@ebi.ac.uk.

MANE Clinical Future project

  • Will aim to contain loci and transcripts that are relevant for clinical testing, including all the transcripts in MANE Plus Clinical.
  • This project is on hold until there is more certainty as to which genes and transcripts are implicated in human disease, which will be done in consultation with clinical experts.

MANE Plus Future project

  • An expansion of MANE transcripts beyond MANE Select and those required for clinical reporting (MANE Plus Clinical) to include transcripts that have additional well-supported features not found in other MANE transcript sets.
  • The MANE Plus project is on hold while EMBL-EBI and NCBI work to improve their independent pipelines. We will:
    • explore and agree on thresholds for more automated methods for a transcript to be included
    • mature our workflows to integrate long transcriptomic data from PacBio and Oxford Nanopore Technologies (ONT)
    • update pipelines to utilise new data arising from the rapid technical development in the wider transcriptomic field (e.g. 5’ capping for PacBio and ONT, tagged full length ONT, direct RNA sequencing from ONT, plus increases in read depth/length/quality).
  • Note: in expanding to MANE Plus, MANE Select transcripts are expected to be stable (although we reserve the right to update them on the rare occasion that new data demonstrate the original determination is incorrect).
